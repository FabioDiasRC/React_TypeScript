------------Adendo - Typescript com React-------------------
Deve-se passar um array em item, caso contrario ele nao aceitaria diversos objetos, aceitaria apenas um, ficando:

  const [list, setList] = useState<Item>(
    {id:1, name: 'tarefa1', done: false},
  )

passando o type de Item como um array, dentro do array passando os items da lista como objetos:

  const [list, setList] = useState<Item[]>([
      {id:1, name: 'tarefa1', done: false},
      {id: 2, name: 'tarefa 2', done: false}
    ])

Sempre separe seus componentes para melhor organização do codigo como um todo, essa é a boa pratica a ser seguida em react. Como ja foi deifnido em 'item.ts' os tipos de cada variavel nao se repete, importa-se para o arquivo e aplica-se assim: 

  import * as C from './style'
  import { Item } from '../../types/item'

  type Props = {
    item: Item
  }

  const ListItem = ({item}: Props) => {
    return (
      <C.Container>
        {item.name}
      </C.Container>
    )
  }

  export default ListItem;

importante reaasaltar que a utiliazação desse item permite ja a importação direta para dentro do container.
